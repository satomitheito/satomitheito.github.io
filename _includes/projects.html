Decoding Song Success

Featured Skills

  • Web Scraping
  • Web APIs
  • Python
  • Natural Language Processing
  • Data Visualization
  • Machine Learning
  • Supervised & Unsupervised Learning
  • Sentiment Analysis

Decoding Song Success leverages data science to provide actionable insights for the music industry. By analyzing data from the Rolling Stones Top 100 Songs, this project bridges NLP and song metadata to uncover factors driving hit songs. Using techniques such as sentiment analysis and predictive modeling, it evaluates how lyric sentiment, artist popularity, and genre influence success.

This project aims to identify key trends in music popularity. It offers tools to predict the commercial potential of songs, informs investment decisions, and helps songwriters and producers refine their creative strategies. From creating genre-specific playlists using unsupervised learning to exploring the impact of collaborations on song success, the findings can optimize decision-making and drive innovation in the music business.

JANAS: Japanese and American News Analysis Sentiment

  • Featured Skills
    • Web Scraping
    • Python
    • Natural Language Processing
    • Multilingual Sentiment Analysis
    • Deep Learning
    • Hugging Face
    • NLTK

    JANAS is a package for multilingual sentiment analysis of Japanese and American news articles. By analyzing trending topics across categories like politics, business, and global news, JANAS offers insights into the differences and similarities in media portrayals between the US and Japan. This project combines web scraping, machine translation, and deep learning-based sentiment analysis to provide a dataset and tools for real-time analysis. The aim is to enable users to explore cultural and linguistic differences in news sentiment and foster cross-cultural understanding.

    Living in Risk: Public Health and Housing Disparities

    Featured Skills

    • Data Analysis
    • Public Health Metrics
    • Housing Risk Analysis
    • Visualization Tools
    • R

    This project investigates the relationship between public health metrics and housing risk scores across various districts. By analyzing vaccination rates, death rates, uninsured percentages, and housing-related factors, the study highlights disparities in housing risks and identifies potential areas for policy intervention. The research explores how public health and socioeconomic indicators correlate with high housing risk scores, aiming to uncover key factors driving these disparities. Through data-driven analysis, the project seeks to provide actionable insights for policymakers to address housing and public health inequalities, prioritizing district-level interventions that promote equitable outcomes.
